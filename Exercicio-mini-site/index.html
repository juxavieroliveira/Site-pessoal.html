Por que escolhi estudar Front-End ?

Sempre tive curiosidade sobre como os sites e aplicativos funcionam, especialmente as partes visuais e interativas. Quando descobri que o front-end é responsável por isso, fiquei muito interessada.

Gosto de ver o resultado do meu trabalho na tela, testar interações e entender como os usuários navegam. Com HTML, CSS e JavaScript, é possível transformar ideias em interfaces reais — e isso me motiva a continuar aprendendo todos os dias.

Meu primeiro projeto em HTML e CSS

Estou nos meus estudos iniciais em front-end, e este é o meu primeiro site pessoal usando apenas HTML5 e CSS3. O objetivo é aprender a estruturar páginas com tags semânticas e aplicar estilos básicos.

Estou entendendo a importância de uma boa estrutura, acessibilidade e responsividade. Também comecei a usar o GitHub para versionar e publicar meu código.
